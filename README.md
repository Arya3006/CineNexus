# 🎬 CineNexus

A full-stack movie ticket booking web application with real-time seat selection and Razorpay payment integration.

---

## 🚀 Features

- 🎬 Browse movies using TMDB API  
- 🪑 Real-time seat selection system  
- 💳 Secure Razorpay payment integration  
- 🧑‍💼 Admin dashboard to manage shows  
- 📋 Booking management system  
- 🔐 User authentication using Clerk  

---

## 🛠 Tech Stack

- **Frontend:** React (Vite), Tailwind CSS  
- **Backend:** Node.js, Express  
- **Database:** MongoDB Atlas  
- **Authentication:** Clerk  
- **API:** TMDB (The Movie Database)  
- **Payment:** Razorpay
